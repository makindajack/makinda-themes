# Makinda Themes — Slack

Custom sidebar themes for the Slack desktop client.

## Install

1. Slack → **Preferences → Themes → Customize my theme…**
2. Paste either the Light or Dark color string from `sidebar-themes.txt`.

Slack only exposes 8 colors for the sidebar; the rest of the chrome follows
your OS appearance.

## Source of truth

Generated from `source/palette.json`. Regenerate with `npm run build:slack`.
