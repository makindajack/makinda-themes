# Makinda Themes — Ghostty

## Install

Copy both files into Ghostty's themes directory:

- macOS: `~/Library/Application Support/com.mitchellh.ghostty/themes/`
- Linux: `$XDG_CONFIG_HOME/ghostty/themes/`

Then in your Ghostty config:

```
theme = Makinda Dark
# or
theme = Makinda Light
```

## Source of truth

Generated from `source/palette.json`. Regenerate with `npm run build:ghostty`.
