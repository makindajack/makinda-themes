# Makinda Themes — Visual Studio (full IDE)

`.vssettings` files for Microsoft Visual Studio (not VS Code).

## Install

1. **Tools → Import and Export Settings…**
2. Choose **Import selected environment settings**.
3. Pick `Makinda Light.vssettings` or `Makinda Dark.vssettings`.

## Source of truth

Generated from `source/palette.json`. Regenerate with `npm run build:visual-studio`.
