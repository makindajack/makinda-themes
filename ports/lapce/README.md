# Makinda Themes — Lapce

## Install

Drop both files into Lapce's themes folder:

```
~/.config/lapce-stable/themes/
```

(or `~/Library/Application Support/lapce-stable/themes/` on macOS).

Then **Settings → Theme → Makinda Light / Makinda Dark**.

## Source of truth

Generated from `source/palette.json`. Regenerate with `npm run build:lapce`.
