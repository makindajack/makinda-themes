## Install

Double-click each `.bbcolors` file in Finder ‚Äî BBEdit will install it. Or
manually copy into:

```
~/Library/Application Support/BBEdit/Color Schemes/
```

Then activate via **BBEdit ‚Üí Preferences ‚Üí Editor ‚Üí Text Colors**.
