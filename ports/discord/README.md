# Makinda Themes — Discord (BetterDiscord)

CSS themes for [BetterDiscord](https://betterdiscord.app/).

## Install

Drop both `.theme.css` files into BetterDiscord's themes directory:

- macOS: `~/Library/Application Support/BetterDiscord/themes/`
- Linux: `~/.config/BetterDiscord/themes/`
- Windows: `%APPDATA%\BetterDiscord\themes\`

Enable via **User Settings → Themes**.

## Source of truth

Generated from `source/palette.json`. Regenerate with `npm run build:discord`.
